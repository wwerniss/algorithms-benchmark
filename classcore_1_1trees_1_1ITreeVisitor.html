Detailed Description

template<typename T>
class core::trees::ITreeVisitor< T >

Interface for the Visitor design pattern applied to binary trees.

Allows defining new operations on tree nodes (like serialization, counting, evaluation) without modifying the internal structure of the tree nodes themselves.
